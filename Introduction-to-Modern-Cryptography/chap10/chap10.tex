\begin{exercise}{10.4}
    By the protocol, it is easy to see 
    $$
    w=u\oplus r=s\oplus t\oplus r=k\oplus r\oplus t\oplus r=k\oplus t,
    $$
    thus $k=w\oplus t$, which means Alice and Bob output the same key.

    This protocol is not secure, since 
    $u\oplus w\oplus s=u\oplus u\oplus r\oplus k\oplus r=k$, which means an eavesdropper 
    can easily obtain the key from the communication.
\end{exercise}
